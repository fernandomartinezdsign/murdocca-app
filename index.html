📋 Guía de Configuración Google Sheets

Paso 1: Ir a Google Sheets y crear una planilla nueva llamada "Murdocca DB".

Paso 2: Ir a Google Cloud Console, crear un proyecto y habilitar la API de Google Sheets.

Paso 3: Crear una API Key y copiarla en el campo de arriba.

Paso 4: Copiar el ID de tu planilla (está en la URL entre /d/ y /edit) y pegarlo arriba.

Paso 5: En tu planilla, hacer la hoja pública (Compartir → Cualquiera con el enlace) para lectura.

Paso 6: Hacer clic en "Probar Conexión". ¡Listo!
